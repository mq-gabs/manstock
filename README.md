# Manstock

## What is it?
This is an application to manage products of a super market/store and register purchases.

## How it works?
- Create account using email/nickname and password
- Register products
- View products
- Register purchases
- View purchases

## What is it made of?
### Frontend
- ReactJS
- Styled Components
### Backend
- Express
- Knex
- Sqlite

## How can I use it?
Just clone this repository and run:
```
docker compose up
```
